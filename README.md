# 100 Java Programs

Java solutions for 100 placement preparation programming problems.

The repository contains one combined Java file, `AllSolutions.java`, with all
100 solved problems written as separate methods from `problem1()` to
`problem100()`.

## Contents

- `AllSolutions.java` - all Java solutions in a single class
- `100_Program_List.pdf` - original list of programming questions
- `.vscode/` - VS Code run/debug configuration

## Topics Covered

- Basic number programs
- Arrays and strings
- Recursion
- Searching and sorting
- Linked lists
- Stacks and queues
- Binary search trees
- Graph traversal and shortest paths
- Dynamic programming
- Greedy algorithms
- Matrix and pattern problems

## How to Run

Compile the Java file:

```bash
javac AllSolutions.java
```

Run the program:

```bash
java AllSolutions
```

By default, `main()` calls all 100 problem methods one by one.

To run a specific problem, edit `main()` in `AllSolutions.java` and call only
the required method, for example:

```java
public static void main(String[] args) {
    problem1();
}
```

## Requirements

- Java JDK 8 or later

Check your Java installation:

```bash
java -version
javac -version
```
